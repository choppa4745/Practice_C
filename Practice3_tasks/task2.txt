В каком сегменте ELF-файла будет находиться global_var?
0000000000000004 C _uninitialized_global_var
global_var будет храниться в сегменте .data, т.к. это инициализированная глобальная переменная

В каком сегменте ELF-файла будет храниться uninitialized_global_var?
0000000000000004 C _uninitialized_global_var
uninitialized_global_var будет храниться в сегменте .bss, т.к. это неинициализированная глобальная переменная

В каком сегменте находится static_var?

Где будет размещена global_point?
00000000000000a4 D _global_point
global_point будет храниться в сегменте .data, т.к. это неинициализированная глобальная структура

Где будут расположены:
local_var из функции print_message?
local_var будет храниться в сегменте .stack, т.к. это локальная переменная функции

static_local_var из той же функции?
00000000000000ac d _print_message.static_local_var
static_local_var будет храниться в сегменте .data, т.к. это инициализированная статическая локальная переменная. Она сохраняет своё значение между вызовами функции.

Где будет храниться auto_var из функции main?
auto_var будет храниться в сегменте .stack, т.к. это локальная переменная функции.

Где будет выделена память для dynamic_var?
dynamic_var будет храниться в сегменте .heap, т.к. память выделяется динамически с помощью malloc.

Где будет создана stack_point?
stack_point будет создана в сегменте .text, т.к. код функции находится в секции .text.
